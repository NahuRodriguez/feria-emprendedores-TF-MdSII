# Feria de Emprendedores

Sistema para gestionar una feria de emprendedores.

## Tecnologías

- Java 17 / Maven
- JUnit 5 + Mockito
- JaCoCo
- GitHub Actions

## Compilar el proyecto

```bash
mvn clean compile
```

# Ejecutar Programa

```bash
java -cp target/classes com.feria.Main
```

# Realizar Tests

```bash
mvn test
```

# Cobertura (reporte en target/site/jacoco/index.html)

```bash
mvn clean verify
```
